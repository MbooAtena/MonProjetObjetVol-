NOMS: MBO'O ATENA SIDONIE ORNELLA
MATRICULE: 24P749

Le projet donne la possibilité a un utilisateur de faire une recherche pour savoir si un objet qu'il souhaiterait avoir , et selon
le statut il saura si c'est un objet volé ou pas(si il est déclaré comme trouvé sur la plateforme)

la base de données ici est une liste où on a préalablement enregistré les objets volés
l'interface est réalisé avec JavaSwing
Exemple pour un potentiel test : Iphone 12, couleur noir 
                                 Mackbook pro , couleur grise
